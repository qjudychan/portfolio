IDEATION

Sketches → Wireframes

Search / Discover

Specify a destination or take inspiration and browse a trending locale (in which case specifics (e.g., travel dates) can be defined later on)

Larger, more legible text in lieu of thumbnails fits more destination options without the need for scrolling


Itinerary Catalog

Build an itinerary by selecting from presented popular attractions at the chosen destination

Option to add itinerary items by address allows for customization for more esoteric interests


Itinerary Cart

Rank items in one's itinerary cart based on importance

Preferences as indicated in this step factor into how information is presented about potential hotels


Custom Map

Hotels are presented based on proximity to items in the user’s itinerary cart but can be narrowed down through filtration


Map Filtration

Filtration is "live" - the number of hotel options updates every time the user edits criteria

After selection of preferences, users can “uncheck” one of the filters to easily see what options open up if they loosen up on a certain criteria


Hotel Detail

How far a hotel is from itinerary items is prominently presented in order to mitigate overlooking this consideration, which normally would have to be researched separately
